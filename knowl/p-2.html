So what if we have an equation like this?
\begin{gather*} f(x)\frac{d^2y}{dx^2} + g(x)\frac{dy}{dx} + h(x) = c? \end{gather*}
It’s still an equation, so we’re still interested in what makes it true. However, the assumption has changed. We don’t need a numerical value to make this true, we need some differentiable function \(y(x)\text{.}\) Conceptually, there isn’t a difference, just procedurally. We’re still trying to find a way to make a statement containing unknown values true.
